# gh-pages 업로드 잘 안되시는 분들 참고하세요

1. repository를 Private -> Public으로 바꾼다.
2. VScode의 view > Terminal를 열고, 터미널에 아래 명령어를 입력한다.

git commit --allow-empty -m "Trigger rebuild"
git push

3. 조금 기다린 후 재시도 한다
